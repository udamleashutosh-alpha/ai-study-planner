📚 AI Study Planner 🚀

«An intelligent productivity tool that helps students plan smarter, stay consistent, and achieve better results using AI-inspired logic.»

---

🔗 Live Demo

👉 https://udamleashutosh-alpha.github.io/ai-study-planner

---

🧠 Problem

Students preparing for exams like JEE struggle with:

- ❌ Poor time management
- ❌ No structured study plan
- ❌ Lack of consistency & motivation

---

💡 Solution

AI Study Planner provides:

- Smart subject-wise time distribution
- AI-based study tips
- Progress tracking system
- Gamified leaderboard

---

✨ Features

- 🔐 Login system (user-based experience)
- 🤖 AI-powered study plan generator
- 💡 Dynamic smart study tips
- 📊 Subject-wise time allocation
- ✅ Progress tracking (checkbox system)
- 📈 Animated progress bar
- 🏆 Leaderboard (gamified learning)
- 📱 Mobile app-like UI
- ✨ Smooth animations using Anime.js

---

🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Anime.js (animations)
- LocalStorage (data persistence)

---

⚙️ How It Works

1. User logs in
2. Enters available study hours
3. AI generates smart study plan
4. Tips + performance mode shown
5. User tracks progress
6. Leaderboard updates automatically

---

🎯 Impact

- Improves daily study consistency
- Helps students plan effectively
- Makes studying engaging via gamification
- Provides structured approach to preparation
